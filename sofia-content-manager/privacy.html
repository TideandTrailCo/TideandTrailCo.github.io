Privacy Policy

App name: Sofia Content Manager

Last updated: May 13, 2026

Overview

Sofia Content Manager ("the App") is a content management platform for independent content creators. It enables creators to automate video publishing to social media platforms via official APIs, including the Content Posting API. This Privacy Policy explains how the App handles data collected through the OAuth authorization flow.

Data We Collect

Sofia Content Manager collects only the following data, obtained through the OAuth authorization flow:

  • User ID (open_id) — used solely to identify the authorized creator account for posting
  • Access token and refresh token — stored securely and used only to authenticate API requests on behalf of the creator
  • Basic profile information (user.info.basic scope) — used to confirm account identity

We do not collect names, email addresses, follower data, video analytics, or any other personal information beyond what is required to authenticate and post content.

How We Use Data

Data collected by Sofia Content Manager is used exclusively to publish video content to the authorized creator account. It is never shared with third parties, sold, or used for advertising, analytics, or any purpose other than content posting.

Data Storage

All authentication tokens are stored securely on the creator's own device or server. Sofia Content Manager does not transmit data to external servers beyond what is required to communicate with the official platform APIs.

Third-Party Services

Sofia Content Manager communicates only with official platform APIs. No other third-party services receive user data.

User Rights

Creators may revoke Sofia Content Manager's access to their account at any time through the platform's Connected Apps settings. Upon revocation, all stored tokens are invalidated.

Children's Privacy

Sofia Content Manager is not intended for use by individuals under the age of 18 and does not knowingly collect data from minors.

Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be reflected by updating the "Last updated" date above.
